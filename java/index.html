What is Java?

Java is a computer programming language that is concurrent, class-based, object-oriented, and specifically designed to have as few implementation dependencies as possible. The principle idea behind Java's design is being able to run on any platform that has the Java Virtual Machine (JVM) installed.

How do I learn Java or any other programming language?

Many people initialy find programming very confusing to learn. The main reason is that when you program, you need to think like a computer. Once you understand the computational thinking that programming intails, it becomes much easier. If you are struggling to learn Java, you could try using Scratch which lets you learn the most important concepts while not having to worry about the syntax since you just drag and drop blocks.

Once you learn the bascis, make sure to try and work on a simple project (such as a basic calculator, number guessing game, or rock paper scissors) in order to reinforce concepts. A great place to go if you are struggling and need help is StackOverflow.

For Loop

    System.out.println("i is " + i);
}

// Loops through items in a list
int[] numbers = {10, 23, -2, 43};
for (int i=0; i < numbers.length(); i++) {
    System.out.println(numbers[i]);
}

// This also loops through items in a list but is more efficient
String[] food = {"Taco", "Pizza", "Chicken", "Salad"};
for (String item : food) {
    System.out.println(item);

public, private, static, and void

| Keyword | Purpose | | |---------|------------------------------------------------------------| |public |Other classes can access the field (variable/method) | |private |Other classes can't access the field | |static |You don't need an instance of the class to access the field | |void |The method doesn't return a value |

public class ClassOne {
    public static void main(String[] args) {
        // ----- ClassTwo -----
        ClassTwo secondClass = new ClassTwo();
                System.out.println(secondClass.intOne); // This will print 2
                System.out.println(secondClass.intTwo); // This will throw an error because intTwo is private   
                System.out.println(secondClass.addNum(1, 2));

        // ----- ClassThree -----
                // Notice how you don't need to type ClassTwo secondClass = new ClassTwo() to access the classes fields
        System.out.println(ClassThree.motorPort);
                ClassThree.sayHi();
    }
}

// ClassTwo doesn't use the static modifier so you must create 
// an instance of the class inorder to access its variables and methods (ClassTwo name = new ClassTwo())
public class ClassTwo {
    public int intOne = 2;
    private int intTwo = 3;

        public void sayHi() {
            System.out.println("Hi!");
        }

    public int addNum(int numOne, int numTwo) {
        return numOne + numTwo;
        }
}

// ClassThree uses the static modifier so you are able to access
// its methods and fields without creating an instance of the class
public class ClassThree {
    public static int motorPort = 2;
        public static String playerName = "Bob";

        public static void sayHi() {
        System.out.println("Hi");   
    }
}
